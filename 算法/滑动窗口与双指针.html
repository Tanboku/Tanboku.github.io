# 定长滑动窗口

首先是定长滑动窗口，下面是一个定长滑动窗口的模板，大部分定长滑动窗口的题都可以用下面这个模板

前提：窗口大小不变，要枚举所有长度恰好为k的区间，其中符合要求的元素有几个

窗口右端点在 i 时，由于窗口长度为 k，所以窗口左端点为 i−k+1。

三步：入-更新-出。

入：下标为 i 的元素进入窗口，更新相关统计量。如果窗口左端点 i−k+1<0，则尚未形成第一个窗口，重复第一步。 更新：更新答案。一般是更新最大值/最小值。 出：下标为 i−k+1 的元素离开窗口，更新相关统计量，为下一个循环做准备。

# 定长子串中元音的最大数目 (opens new window)

套模板就好

class Solution {
    public static final String VOWEL = "aeiou";

    public int maxVowels(String s, int k) {
        int ans=0;//答案
        int value=0;//相关统计量
        char[] c = s.toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<s.length();i++){//遍历
            //入
            char in = c[i];//入队元素

            //相关统计量
            if(VOWEL.indexOf(in)!=-1)value++;//满足条件就统计

            //判断是否形成窗口,left为窗口长度
            int left = i-k+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            ans = Math.max(ans,value);//根据题目要求判断

            //出
            char out = c[left];//出队元素
            if(VOWEL.indexOf(out)!=-1)value--;//满足条件就去掉
        }
        return ans;//返回答案
    }
}

# 子数组最大平均数 I (opens new window)

模板题

class Solution {
    public double findMaxAverage(int[] nums, int k) {
        double ans=Integer.MIN_VALUE;//答案
        double value=0;//相关统计量
        //char[] c = .toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<nums.length;i++){//遍历
            //入
            int in = nums[i];//入队元素

            //相关统计量
            value+=in;//满足条件就统计

            //判断是否形成窗口,left为窗口长度
            int left = i-k+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            ans = Math.max(ans,value/k);//根据题目要求判断

            //出
            int out = nums[left];//出队元素
            value-=out;//满足条件就去掉
        }
        return ans;//返回答案
    }
}

# 大小为 K 且平均值大于等于阈值的子数组数目 (opens new window)

模板题

class Solution {
    public int numOfSubarrays(int[] arr, int k, int threshold) {
        int ans=0;//答案
        int value=0;//相关统计量
        //char[] c = .toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<arr.length;i++){//遍历
            //入
            int in =arr[i];//入队元素

            //相关统计量
            value+=in;//满足条件就统计
            

            //判断是否形成窗口,left为窗口长度
            int left = i-k+1;
            if(left<0)continue;//如果没有形成就重复第一步

            ans += (value/k)>=threshold?1:0;//根据题目要求判断

            //出
            int out = arr[left];//出队元素
            value-=out;//满足条件就去掉
            
        }

    }

# 半径为 k 的子数组平均值 (opens new window)

模板题，不过要注意实际窗口大小和数据范围

class Solution {
    public int[] getAverages(int[] nums, int k) {
        int[] ans= new int[nums.length];//答案
        long value=0;//相关统计量
        Arrays.fill(ans,-1);
        int len = 2*k+1;
        //char[] c = .toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<nums.length;i++){//遍历
            //入
            int in = nums[i];//入队元素

            //相关统计量
            value+=in;//满足条件就统计

            //判断是否形成窗口,left为窗口长度
            int left = i-len+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            ans[i-k] = (int)(value/len);//根据题目要求判断

            //出
            int out = nums[left];//出队元素
            value-=out;//满足条件就去掉
        }
        return ans;//返回答案
    }
}

# 得到 K 个黑块的最少涂色次数 (opens new window)

模板题，翻译一下就是找到定长字串中W字符的最小数目

class Solution {
    public int minimumRecolors(String blocks, int k) {
        int ans= Integer.MAX_VALUE;//答案
        int value=0;//相关统计量
        char[] c = blocks.toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<blocks.length();i++){//遍历
            //入
            char in = c[i];//入队元素

            //相关统计量
            if(in == 'W'){
                value++;//满足条件就统计
            }

            //判断是否形成窗口,left为窗口长度
            int left = i-k+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            ans = Math.min(ans,value);//根据题目要求判断

            //出
            char out = c[left];//出队元素
            if(out=='W'){
                value--;//满足条件就去掉
            }
        }
        return ans;//返回答案
    }
}

# 几乎唯一子数组的最大和 (opens new window)

模板题，注意map是怎么用于判断的

class Solution {
    public long maxSum(List<Integer> nums, int m, int k) {
        long ans=0;//答案
        long value=0;//相关统计量
        //char[] c = .toCharArray();//字符串转成字符数组，方便遍历
        Integer[] a = nums.toArray(Integer[]::new);//集合转成数组，方便遍历
        Map<Integer,Integer> map = new HashMap<>();
        for(int i =0;i<a.length;i++){//遍历
            //入
            int in = a[i];//入队元素
            map.merge(in,1,Integer::sum);

            //相关统计量
            value+=in;//满足条件就统计

            //判断是否形成窗口,left为窗口长度
            int left = i-k+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            if(map.size()>=m){
                ans = Math.max(ans,value);//根据题目要求判断
            }
            
            //出
            int out = a[left];//出队元素
            value-=out;//满足条件就去掉
            if(map.get(out)>1){
                map.merge(out,-1,Integer::sum);
            }else{
                map.remove(out);
            }
        }
        return ans;//返回答案
    }
}

# 可获得的最大点数 (opens new window)

模板题，脑筋急转弯，将卡牌分数更改为总数减去滑动窗口的分数就是经典定长滑动窗口的题

class Solution {
    public int maxScore(int[] cardPoints, int k) {
        int ans=0;//答案
        int value=0;//相关统计量
        int len = cardPoints.length-k;
        int sum =0;
        for(int num:cardPoints){
            sum+=num;
        }
        if(len ==0)return sum;
        //char[] c = .toCharArray();//字符串转成字符数组，方便遍历
        //Integer[] a = .toArray(Integer[]::new);//集合转成数组，方便遍历
        for(int i =0;i<cardPoints.length;i++){//遍历
            //入
            int in =cardPoints[i];//入队元素

            //相关统计量
            value+=in;//满足条件就统计

            //判断是否形成窗口,left为窗口长度
            int left = i-len+1;
            if(left<0)continue;//如果没有形成就重复第一步

            // 更新答案
            ans = Math.max(ans,sum-value);//根据题目要求判断

            //出
            int out = cardPoints[left];//出队元素
            value-=out;//满足条件就去掉
        }
        return ans;//返回答案
    }
}
