# 编程初学者的新手村

视频讲解：力扣算法记录-编程入门_哔哩哔哩_bilibili (opens new window)

首先学会根据数据范围估计使用什么算法，一般每秒约能执行 10^8^ 次运算，可以据此估计能通过的时间复杂度，如下表所示。

数据范围 允许的时间复杂度 适用算法举例
n≤10 O(n!) 或 O(C^n^) 回溯、暴力搜索
n≤20 O(2*^n^*) 状态压缩 DP
n≤40 O(2^n/2^) 折半枚举
n≤10^2^ O(n^3^) 三重循环的 DP、Floyd
n≤10^3^ O(n^2^) 二重循环的 DP、背包
n*≤10^5^ O(nlogn) 大多数题目都是这个范围，所以各类算法都有
n≤10^6^ O(n) 线性 DP、滑动窗口
n≤10^9^ O(n^1/2^) 判断质数
n≤10^18^ O(logn) 或 O(1) 二分、快速幂、数学公式

注：实际做题时，注意常数因子的影响。例如哈希表比数组慢。

# 最小偶倍数 (opens new window)

给一个正整数n，返回 2 和 n 的 最小公倍数。

判断 n 是不是偶数，如果是偶数就说明最小公倍数是 n 本身，是奇数就乘二，这里我用了三元运算符的写法，(条件表达式) ? 表达式1 : 表达式2，条件为真就返回表达式1，条件为假就返回表达式二。

class Solution {
    public int smallestEvenMultiple(int n) {
        return n%2==0?n:n*2;
    }
}

也可以通过正常方法计算最小公倍数，通过这道题，还可以掌握最大公约数和最小公倍数的计算方法：

class Solution {
    public int smallestEvenMultiple(int n) {
        return lcm(2,n);
    }

    private int lcm(int a, int b){
        return a/gcd(a,b) * b;
    }

    private int gcd(int a,int b){
        while(b!=0){
            int t = a%b;
            a = b;
            b = t;
        }
        return a;
    }
}

# 判断根结点是否等于子结点之和 (opens new window)

两整数相加二叉树版本，加个条件判断就行。

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public boolean checkTree(TreeNode root) {
        return root.val == (root.left.val + root.right.val);
    }
}

# 数组异或操作 (opens new window)

很简单的模拟题，按照题目要求循环模拟即可.

这里可以说下异或操作，在位运算中，两个数不同为1，相同为0，比如1101 ^ 0111 = 1010

class Solution {
    public int xorOperation(int n, int start) {
        int ans =0;
        for(int i =0; i<n; i++){
            ans ^= start + 2*i;
        }
        return ans;
    }
}

# 好数对的数目 (opens new window)

记录每一次每一种数出现的次数，举个例子，1这个数字已经出现过两次，再次遇到1后这个1可以和前面遇到过的每一个1都组成一个好数对，也就是加上出现过的次数。 那么思路就很明显了，循环遍历，遇到一个数就将对应数据出现的次数先与答案相加再加一，如果没遇见过就为0，最后返回答案就行。

这里引入一个很好用的数据结构，哈希。

哈希表是键值对结构，一个 key 对应一个 value ，比如我将 key 设置为 1，value为 3，就代表1这个数字出现过三次。

class Solution {
    public int numIdenticalPairs(int[] nums) {
        Map<Integer,Integer> map = new HashMap<>();
        int ans =0;
        for(int num:nums){
            ans += map.getOrDefault(num,0);
            map.merge(num,1,Integer::sum);
        }
        return ans;
    }
}

# 统计好三元组 (opens new window)

也是一个简单的循环模拟，三层循环内判断条件计入答案就行。

class Solution {
    public int countGoodTriplets(int[] arr, int a, int b, int c) {
        int n = arr.length;
        int ans =0;
        for(int i =0;i<n-2;i++){
            for(int j =i+1;j<n-1;j++){
                for(int k =j+1;k<n;k++){
                    if(
                        Math.abs(arr[i]-arr[j])<=a && 
                        Math.abs(arr[j]-arr[k])<=b && 
                        Math.abs(arr[i]-arr[k])<=c
                        ){
                            ans++;
                        }
                }
            }
        }
        return ans;
    }
}

class Solution {

        char[] t = s.toCharArray();

class Solution {

        char[] t = s.toCharArray();
        for (int i = 0; i < t.length; i++) {
            char ch = t[i];
            if ('A' <= ch && ch <= 'Z') {
                t[i] |= 32;
            }
        }
        return new String(t);
    }
}

# 各位相加 (opens new window)

这道题本质是计算自然数 num 的数根 (opens new window)。

将一正整数的各个位数相加(即横向相加)后，若加完后的值大于等于10的话，则继续将各位数进行横向相加直到其值小于十为止所得到的数，即为数根。换句话说，数根是将一数字重复做其数字之和，直到其值小于十为止，则所得的值为该数的数根。例如54817的数根为7，因为5+4+8+1+7=25，25大于10则再加一次，2+5=7，7小于十，则7为54817的数根。x+9与x的数根相同，即一个数加9后它的数根不变。

所以我们就有了一个求数根的公式：a的数根b = (a-1) % 9+1, 即 mod(a-1, 9)+1，且a ∈ N*

class Solution {
    public int addDigits(int num) {
        return (num -1)%9+1;
    }
}

# 整数的各位积和之差 (opens new window)

枚举，计算，返回答案

class Solution {
    public int subtractProductAndSum(int n) {
        int prod = 1;
        int sum = 0;
        while (n > 0) {
            prod *= n % 10;
            sum += n % 10;
            n /= 10;
        }
        return prod - sum;
    }
}

# 2 的幂 (opens new window)

如果一个数是2的n次幂，那么，这个数换算成二进制，一定是最左边是1，其他位是0的形式 比如4，它的二进制是100，8的二进制是1000 n-1的二进制，一定是n的每一位取反，也就是1变成0，0变成1 比如7的二进制是0111，3的二进制是011.

因此，通过 n > 0 且 n & (n - 1) == 0 即可判定是否满足 n=2^x^。

class Solution {
    public boolean isPowerOfTwo(int n) {
        return n > 0 && (n & (n - 1)) == 0;
    }
}

# 3 的幂 (opens new window)

3 的幂一定大于 0。如果 n≤0，那么 n 不是 3 的幂。下文假定 n>0。在本题的数据范围下，最大的 3 的幂是 3^19^=1162261467。

如果 n 是 3 的幂。由于任何 3 的幂都是 3^19^的因子，所以 3^19^ mod n=0。或者说 n 的某个倍数等于 3^19^ 。 如果 n 不是 3 的幂。那么 n 必然有不等于 3 的质因子，所以 n 的倍数不可能等于 3^19^这个只含质因子 3 的数，所以 3^19 ^mod n != 0。 因此，在本题数据范围下，当且仅当 n>0 且 3^19^ mod n=0 时，n 是 3 的幂。

class Solution {
    public boolean isPowerOfThree(int n) {
        return n > 0 && 1162261467 % n == 0;
    }
}

# 丑数 (opens new window)

暴力解法就是排除非正整数后，2，3，5这三个数挨个除到不能整除位置，最后如果 n == 1，就说明是丑数，如果要优化可以只除 3 和 5 ，最后用2 的幂 (opens new window)的方法判断数字是不是 2 的幂就行。

class Solution {
    public boolean isUgly(int n) {
        if(n<=0)return false;
        while(n%2==0){
            n/=2;
        }
        while(n%3==0){
            n/=3;
        }
        while(n%5==0){
            n/=5;
        }
        return n==1;
    }
}

# 重新排列数组 (opens new window)

创建一个二倍大的新数组，然后将数据挨个填进去。

class Solution {
    public int[] shuffle(int[] nums, int n) {
        int[] ans = new int[n * 2];
        for (int i = 0; i < n; i++) {
            ans[i * 2] = nums[i];
            ans[i * 2 + 1] = nums[n + i];
        }
        return ans;
    }
}

也可以挑战下原地移动

class Solution {
    public int[] shuffle(int[] nums, int n) {
        for (int i = 0; i < nums.length; i++) {
            int x = nums[i];
            if (x < 0) {
                continue;
            }
            int cur = i;
            while (true) {
                int nxt = cur < n ? cur * 2 : (cur - n) * 2 + 1;
                if (nxt == i) {
                    nums[i] = -x;
                    break;
                }
                int tmp = nums[nxt];
                nums[nxt] = -x;
                x = tmp;
                cur = nxt;
            }
        }
        for (int i = 0; i < nums.length; i++) {
            nums[i] = -nums[i];
        }
        return nums;
    }
}

# 转置矩阵 (opens new window)

创建一个新的矩阵，转着圈给新数组赋值。

class Solution {
    public int[][] transpose(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        int[][] ans = new int[n][m];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                ans[j][i] = matrix[i][j];
            }
        }
        return ans;
    }
}

# 分割字符串的最大得分 (opens new window)

枚举每个分隔符，先记录一共有多少个1，数量就是分隔符在最左边时的分数，接下来遇到0就加一，遇到1就减一。

class Solution {
    public int maxScore(String S) {
        char[] s = S.toCharArray();
        int res = 0;
        for (char c : s) {
            res += c - '0';
        }

        int ans = 0;
        for (int i = 0; i < s.length - 1; i++) {
            res += s[i] == '0' ? 1 : -1;
            ans = Math.max(ans, res);
        }
        return ans;
    }
}

# 统计范围内的元音字符串数 (opens new window)

class Solution {
    private static final String VOWEL = "aeiou";

    public int vowelStrings(String[] words, int left, int right) {
        int ans = 0;
        for (int i = left; i <= right; i++) {
            String s = words[i];
            if (VOWEL.indexOf(s.charAt(0)) != -1 &&
                VOWEL.indexOf(s.charAt(s.length() - 1)) != -1) {
                ans++;
            }
        }
        return ans;
    }
}

# 山脉数组的峰顶索引 (opens new window)

题目要求必须设计并实现时间复杂度为 O(log(n)) 的解决方案。根据表格看，时间复杂度为 O(log(n))的算法可以使用二分法，二分具体的实现可以到对应专题观看。

class Solution {
    public int peakIndexInMountainArray(int[] arr) {
        int left = 0;
        int right = arr.length - 2;
        while (left + 1 < right) {
            int mid = left + (right - left) / 2;
            if (arr[mid] > arr[mid + 1]) {
                right = mid;
            } else {
                left = mid;
            }
        }
        return right;
    }
}
